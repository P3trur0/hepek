        Prerequisites

        Prerequisites:

        • Java 8 JDK, because Scala 2.12 uses it.
        • sbt

        There is a plan to support other build tools in the future: CBT, Mill, Gradle etc.
        Contributions are welcome.

        Installation

        Download the starter project from here and you're ready to roll.
        Start up sbt console and type hepek.
        Voila, that's it! :)

        HTML/CSS or whatever you're rendering will be in /target/web/public/main/site folder. You can make an sbt task for copying these to a more desirable location.
