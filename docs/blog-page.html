        Blog post settings

        When you extend BlogPostPage you get support for a blog post page.
        For a Bootstrap-themed page extend HepekBootstrap3BlogPage.
        BlogPostPage has the following fields:

        NameTypeMandatoryDefault valueDescription
        postAuthorOption[String]NoNoneAuthor of the post
        postCreateDateOption[LocalDate]NoNoneDate when the post was written
        postSectionsList[Section] NoList.emptySections of the post
        categoryPostsList[BlogPostPage]NoList.emptyRelated posts, from same category

        Sections

        You need to override the postSections method.
        A blog post is made from sections, so you can render a nice TOC (or even a PDF later, why not?).
        Sections are hierarchical, they can contain other sections.
        Every section has a name (title), content, and optionally child sections.
        Code usually looks similar to this:

        
        trait MySiteBlogPostTemplate extends BlogPostPage {
          override def postSections = List(firstSection)

          val firstSection = Section(
            "Hello world!",
            p("Welcome to my blog!"),
            List(innerSection) // child sections, optional
          )
          val innerSection = Section(
            "Goodbye!",
            p("Thanks for visiting!")
          )
        }

        Related posts

        Note the categoryPosts method. It is used for grouping posts, posts that belong to same "category".
        Also, usually you'll want to have a sidebar with related posts links.

        Example: Let's say you are writing math tutorials.
        You'll have separate templates for different topics: SetTheoryTemplate, AlgebraTemplate etc.
        In each template you populate categoryPosts list and you'll get a nice sidebar. :)
